## ❗️ Troubleshooting
| Problem                  | Solution                                      |
|--------------------------|-----------------------------------------------|
| Game doesn't start       | Run as Administrator                          |
| Low performance          | Lower graphics settings                       |
| Black screen             | Switch to Borderless Windowed mode            |
| Antivirus warning        | Add the folder to exclusions                  |
| Controller not working   | Enable controller support in game settings    |
